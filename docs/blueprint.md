# **App Name**: ThreatWatch AV

## Core Features:

- Dashboard Metrics: Display key metrics related to autonomous vehicle threat detection, providing a real-time overview of system performance.
- GPS Spoofing Detection: Enable uploading and analysis of GPS data to detect potential spoofing events.
- Sensor Anomaly Analysis: Facilitate the uploading and analysis of sensor data to identify anomalies and potential security breaches.
- Sybil Attack Detection: Allow users to upload vehicle communication logs to identify potential Sybil attacks within the autonomous vehicle network.
- Risk Assessment: Use a generative AI tool to analyze all threat intelligence signals, and generate a tool summary of the highest priority threat impacting vehicle safety.

## Style Guidelines:

- Primary color: Deep blue (#0B5FFF) to reflect the advanced technology and security focus of the application.
- Background color: Light gray (#F3F4F6) for a clean and neutral backdrop that ensures readability and focus on content.
- Accent color: Vibrant orange (#F59E0B), for interactive elements such as buttons and tags to draw attention to key actions and alerts.
- Font: 'Inter' (sans-serif) for a clean and modern look throughout the dashboard. Note: currently only Google Fonts are supported.
- Code blocks: 'Roboto Mono' for displaying code snippets in the GPS Spoofing Detection page.
- Utilize a 12-column grid for desktop, 8-column for tablet, and 4-column for mobile to maintain responsiveness across different devices.
- Employ auto layout and consistent spacing tokens (12px, 16px, 20px, 24px) to create a cohesive and user-friendly interface.
